🔐 Password Checker

A simple Python-based CLI tool to evaluate the strength of a password based on common security rules.

## Features
* Minimum 8 characters
* Includes numbers
* Includes uppercase & lowercase

## Getting Started
### Prerequisites
* Python 3.x installed

### Installation
* git clone https://github.com/your-username/password-checker.git
* cd password-checker

## Run
python password_checker.py

## Tech
* Python 
* Regex (re)
